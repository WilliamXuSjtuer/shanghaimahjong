    Audio Files Directory Structure
============================================================

【目录结构】

audio/
├── female/                    # 女声语音
│   ├── tiles/                 # 牌语音 (MP3格式)
│   │   └── *.mp3
│   └── actions/               # 动作语音 (MP3格式)
│       └── *.mp3
├── male/                      # 男声语音
│   ├── tiles/                 # 牌语音 (MP3格式)
│   │   └── *.mp3
│   └── actions/               # 动作语音 (MP3格式)
│       └── *.mp3
└── README.txt

【格式要求】
- 统一使用 MP3 格式（兼容性最好）
- 时长：每个 0.5-1 秒
- 采样率：44100Hz 或 22050Hz
- 比特率：128kbps 即可

============================================================
游戏内使用
============================================================

- 玩家可以在游戏开始前选择"男声"或"女声"
- 出牌时会根据出牌玩家的性别设置播放对应的语音
- AI 玩家有固定性别：
  - AI小明：男声
  - AI小红：女声
  - AI小刚：男声
  - AI小丽：女声
